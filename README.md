# 🔐 Python Cryptography Implementations

A cybersecurity project demonstrating the implementation of classical and modern cryptographic algorithms using Python.

## 📌 About The Project

This repository contains practical implementations of different encryption algorithms created to understand the fundamentals of cryptography, data protection, and secure communication.

The project covers encryption, decryption, key generation, and cryptographic concepts commonly used in cybersecurity.

## 🧪 Implemented Cryptography Labs

## 01 - Caesar Cipher 🔤

A classical substitution cipher implementation.

**Features:**

* Text encryption
* Text decryption
* Shift key-based encryption
* Supports alphabetic characters

---

## 02 - DES Encryption 🔒

Implementation of the Data Encryption Standard (DES) algorithm.

**Features:**

* Symmetric encryption
* Random key generation
* Encryption and decryption process
* Hexadecimal ciphertext output

---

## 03 - AES Encryption 🛡️

Implementation of AES encryption with SHA-256 key derivation.

**Features:**

* AES-256 encryption
* Secret key generation
* Data encryption and decryption
* Base64 encoded output

---

## 04 - RSA Encryption 🔑

Implementation of RSA public-key cryptography.

**Features:**

* Public key generation
* Private key generation
* Message encryption
* Message decryption

---

## 05 - RSA Key Generation ⚙️

Implementation of RSA key generation process.

**Features:**

* Prime number generation
* Calculation of n and φ(n)
* Public and private key creation
* RSA mathematical operations

---
